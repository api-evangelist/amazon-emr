# Amazon EMR

Amazon EMR (Elastic MapReduce) is a cloud big data platform for processing vast amounts of data using open-source tools such as Apache Spark, Apache Hive, Apache HBase, Apache Flink, and Presto. EMR makes it simple to set up, operate, and scale big data environments for analyzing and processing data at scale.

## APIs

- **Amazon EMR API** - The Amazon EMR API enables programmatic access to create, manage, and monitor big data clusters running Apache Spark, Hadoop, and other open-source frameworks. You can launch clusters, add processing steps, configure scaling policies, and manage cluster lifecycle.
