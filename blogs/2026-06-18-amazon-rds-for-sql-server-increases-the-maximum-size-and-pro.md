date: "2026-06-18"

Amazon RDS for SQL Server now supports higher volume-level limits for General Purpose (gp3) storage. Each gp3 volume can now scale up to 64 TiB (4X the previous 16 TiB limit), up to 80,000 IOPS (5X the previous 16,000 limit), and up to 2,000 MiB/s throughput (2X the previous 1,000 MiB/s limit).
