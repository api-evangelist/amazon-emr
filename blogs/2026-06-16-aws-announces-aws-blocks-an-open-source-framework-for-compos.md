AWS announces the public preview of AWS Blocks, an open-source TypeScript framework that runs a fully functional local environment with Postgres, authentication, and real-time messaging with no AWS account required. The same application code runs on production AWS services with zero changes, and developers can drop into AWS CDK at any point.
