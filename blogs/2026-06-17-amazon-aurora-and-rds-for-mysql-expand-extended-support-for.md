Amazon Aurora MySQL-Compatible Edition and Amazon RDS for MySQL now offer Extended Support for MySQL 5.7 through June 30, 2029, from the previous end date of February 28, 2027. This applies to Aurora MySQL version 2 and RDS for MySQL version 5.7, giving customers more time to upgrade while continuing to receive critical security patches and bug fixes.
