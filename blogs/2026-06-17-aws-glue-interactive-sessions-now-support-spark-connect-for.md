AWS Glue Interactive Sessions now support Apache Spark Connect, letting developers run Spark applications from notebooks in SageMaker Unified Studio, Jupyter, or VS Code while running them on Glue's serverless infrastructure without managing clusters. Spark Connect uses a thin client architecture that decouples the client application from the Spark execution environment.
