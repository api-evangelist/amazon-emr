AWS Secrets Manager now offers a secret safety skill as part of the aws-core plugin in the Agent Toolkit for AWS, an open-source repository that equips AI coding agents with tools and guardrails for building on AWS. The skill lets developers use secrets within agentic workflows without exposing secret values to the underlying model or session logs.
