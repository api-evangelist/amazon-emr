Customers using Amazon SNS in the Asia Pacific (Seoul) Region can now send text messages (SMS) to subscribers in more than 200 countries and territories. Amazon SNS is a fully managed pub/sub messaging service that enables message delivery to endpoints including AWS Lambda, Amazon SQS, Amazon Data Firehose, mobile devices, and email.
