AWS Config now supports 9 additional resource types across Amazon Bedrock, Amazon Bedrock AgentCore, and Amazon SageMaker, including FlowAlias, Evaluator, GatewayTarget, OnlineEvaluationConfig, RuntimeEndpoint, and SageMaker Cluster, Endpoint, ModelPackageGroup, and Pipeline.
