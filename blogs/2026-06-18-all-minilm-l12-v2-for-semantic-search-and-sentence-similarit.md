AWS announced the availability of all-MiniLM-L12-v2 in Amazon SageMaker JumpStart. This model from Sentence Transformers maps sentences and paragraphs to a 384-dimensional dense vector space, enabling customers to build semantic search, text clustering, and sentence similarity applications on AWS infrastructure.
