AWS HealthOmics now streams workflow engine logs to Amazon CloudWatch in real time, enabling customers to monitor workflow execution progress as it happens. HealthOmics is a HIPAA-eligible service that helps healthcare and life sciences customers run fully managed bioinformatics workflows at scale.
