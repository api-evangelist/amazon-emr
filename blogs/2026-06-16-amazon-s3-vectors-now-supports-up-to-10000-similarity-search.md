Amazon S3 Vectors can now return up to 10,000 similarity search results per query, a 100x increase from the previous limit, helping applications with multi-stage retrieval pipelines apply reranking, aggregations, or deduplication. Results are returned across multiple pages and the first 512 KB of data returned per query is free.
