Web Search on AgentCore is now generally available, a fully managed tool that grounds agent responses in current web knowledge with zero data egress, exposed as a built-in connector target on AgentCore gateway using the Model Context Protocol (MCP). It combines an Amazon-operated web index with structured knowledge graph data and is available in US East (N. Virginia).
